## 💡 About CodeHashers

CodeHashers is a **Decentralized Freelancer Platform** built on blockchain that enables:

* 🔐 Trustless payments via smart contracts
* 🌍 Borderless freelancing
* ⭐ Tamper-proof reputation
* 👤 Wallet-based authentication

---

## ⚙️ Tech Stack

* Frontend: React / Next.js
* Backend: Node.js
* Blockchain: Solidity
* Storage: IPFS

---

## 🔥 Core Features

* 💸 Escrow Payment System
* 🧾 Transparent Transactions
* 📊 Project Management
* 🔗 Web3 Integration

---

## 🧠 Vision

> “To remove intermediaries and build the future of freelancing on blockchain.”
